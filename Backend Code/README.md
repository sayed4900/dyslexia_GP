# Backend Services

This directory contains the backend services powering the Lexia Learn platform. Each service is maintained in its own repository and included here as a submodule for easier development and deployment.

## 📦 Backend Services

| Service | Description | Repository | Tech Stack |
|---------|-------------|------------|------------|
| **Node.js API Server** | Main backend API handling user management, cognitive tests, and data storage | [GP_node_server](https://github.com/sayed4900/GP_node_server) | Node.js, Express, MongoDB, JWT, Cloudinary |
| **Flask Eye-Tracking Server** | Microservice for eye gaze direction prediction using deep learning | [GP_flask_server](https://github.com/sayed4900/GP_flask_server) | Python, Flask, TensorFlow, Keras |
